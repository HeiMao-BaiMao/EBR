# EBR - Project Overview

EBR is a desktop EPUB Reader application built with Tauri 2.
It allows users to manage a local library of EPUB files (scanned from directories) and read them in a customized viewer.

## Key Features
- **Bookshelf:** Scans folders for EPUB files and displays them in a grid.
- **Reader:** Opens EPUBs using `epub.js`. Supports pagination and keyboard navigation.
- **Theming:** Supports Light and Dark modes (system preference or user toggle).
- **Persistent Settings:** Remembers library paths and theme preference.
